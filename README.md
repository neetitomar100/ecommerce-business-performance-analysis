# Ecommerce Business Performance Analysis

## Project Overview
This project focuses on analyzing Ecommerce business performance using **Excel, Power BI, DAX, SQL, and PostgreSQL**.

The objective of this project is to identify key business insights related to:

- Revenue
- Orders
- Product performance
- Profitability
- Refund trends
- Monthly sales performance

This project demonstrates an end-to-end analytical workflow including **raw data handling, SQL analysis, KPI creation, dashboard development, and business insight generation**.

This project follows a complete analytical workflow from **raw Ecommerce data to dashboard reporting**, using multiple tools for data analysis and KPI generation.

---

## Tools Used
- **Microsoft Excel** – Data cleaning and support analysis
- **Power BI** – Dashboard creation and visualization
- **DAX** – KPI calculations and calculated columns
- **SQL** – Data querying and business analysis
- **PostgreSQL** – Structured data analysis and reporting

## Business Questions Solved
This project answers the following business questions:

1. What is the total revenue generated?
2. How many total orders were placed?
3. What is the average order value?
4. Which products are top-selling?
5. Which products generate the highest profit?
6. What is the monthly revenue trend?
7. Which products have the highest refund amount?
8. What is the profit margin by product?
9. Which months generated the highest revenue?
10. What is the refund rate by product?
11. What is the net revenue after refunds?
